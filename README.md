# Wedding Client
- Welcome to the wedding website.

## Features

- View couple time line.

- Meet wedding party personels.

- View photos.

- RSVP only if on the invite list.

## Tech Stack

### Client:
- React.js
- JavaScript
- HTML
- CSS

### Server:
- Node.js
- Express
- Knex
- PostgreSQL
- Heroku


### Local Setup

To start:
```
npm install
npm i react-chrono
```
